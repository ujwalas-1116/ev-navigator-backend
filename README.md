# EV Charging Station Navigator - Backend Service

This is the Express + Node.js API backend service for the EV Charging Station Navigator.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/ujwalas-1116/ev-navigator-backend)

## How to Deploy Online

1. Click the **Deploy to Render** button above.
2. Render will automatically configure the repository and display a setup screen.
3. In the **Environment Variables** section, enter:
   * **Key**: `MONGODB_URI`
   * **Value**: `mongodb+srv://Ujwala-1116:Ujwala1116@cluster0.bm1gdgk.mongodb.net/ev_navigator?retryWrites=true&w=majority&appName=Cluster0`
4. Click **Apply / Deploy**!
